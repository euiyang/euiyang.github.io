# process explorer  
process explorer는 Windows os의 최고의 프로세스 관리 도구이며 다음과 같은 장점이 있다.  
1) parent/child 프로세스 트리 구조  
2) 프로세스 실행/종료 색깔로 표시  
3) 프로세스 suspend/resume 기능  
4) 프로세스 kill 기능  
5) DLL/handle 검색  
등이 있고 리버싱할 때 도움이 될 수 있다.
